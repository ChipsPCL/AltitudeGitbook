# Security Overview

Security is important across the Altitude ecosystem.

The Altitude token contract was audited by CryptoHub in November 2023. The contract remains unchanged since that audit.

Altitude's largest liquidity pool at the time of writing is Altitude/WETH on BaseSwap. This LP was sent to the burn address one year after launch and cannot be accessed.

Mountain Miner is expected to have a SolidityScan Enhanced audit report available soon.

{% hint style="warning" %}
Audits and automated scans reduce risk but do not eliminate it. Users should still treat all DeFi contracts as high risk.
{% endhint %}
