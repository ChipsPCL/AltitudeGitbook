# Roadmap

Altitude's roadmap focuses on building more utility from within the ecosystem while continuing to support the Altitude token.

## Current Roadmap Items

* Altitude Farming Protocol.
* New Altitude/WETH liquidity pool on Aerodrome or Uniswap.
* Explore loan options using Altitude as collateral.
* Launch and support Mountain Miner Season 1.
* Continue development of the Everest USDC Rewards Vault.
* Continue building ecosystem products that can generate external revenue for the Mountain treasury.

## Long-Term Direction

Altitude's long-term goal is to create a self-supporting ecosystem where external revenue, treasury activity, project-owned liquidity, and ecosystem products all help strengthen the Altitude token over time.
